# 🌱 Fasila – Plant E-commerce Website

Fasila is a full-stack e-commerce web application developed as an individual college project.

The platform allows users to browse, purchase, and sell plants through a simple and responsive interface.  
The goal of the project was to practice building a complete web system including frontend, backend, and database integration.

---

## ✨ Key Features

- 🌿 Product browsing and plant catalog
- 🛒 Shopping cart functionality
- 📦 Order management
- 🏪 Seller listing functionality
- 📱 Responsive user interface
